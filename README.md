# Sidebar Menu Project

A simple and responsive sidebar menu built using **HTML** and **CSS**. The project uses the CSS checkbox hack to create a sliding sidebar without JavaScript.

## 🚀 Features

- Sliding sidebar navigation
- Responsive full-screen background image
- Font Awesome icons
- Smooth CSS transitions
- Hover effects on menu and social media icons
- Pure HTML & CSS (No JavaScript)

## 🛠️ Technologies Used

- HTML5
- CSS3
- Font Awesome
